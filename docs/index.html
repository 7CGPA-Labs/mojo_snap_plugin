Retro Console Platform

Play local retro classics anywhere. Completely serverless.

A high-performance emulation platform running vanilla RetroArch WebAssembly cores directly in your browser. Restructured as a monorepo to power cross-browser extensions and media server plugins from a single client codebase.

Download Extension Get Media Plugin

Core Features

⚡

WebAssembly Powered

Runs fceumm, snes9x, and picodrive cores locally inside client browsers with native speed.

🎮

Universal Gamepads

Auto-detects USB and Bluetooth controllers via the standard W3C Gamepad API.

📁

Zero Server Overhead

Load ROM files locally from your system; no game files are hosted or bundled in the package.

⚙️

RetroArch Controls

Remap gamepad buttons, load cheats, and manage save states locally in browser IndexedDB storage.

Quick Installation

Chrome/Edge Extension

  1. Download the repository and run build_extension.ps1.
  2. Open chrome://extensions.
  3. Enable Developer Mode.
  4. Click Load unpacked and select the extension build.

Emby/Jellyfin Plugin

  1. Compile the plugin project using build_plugin.ps1.
  2. Copy the compiled DLL into your server's plugins/ folder.
  3. Restart your media server and view games natively in your library.
